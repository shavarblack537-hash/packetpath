STP RSTP Root Bridge Port States
60–90 min GNS3 / PT

Intermediate Lab 06

The server subnet is wide open — any internal user can reach it. Configure standard and extended ACLs to restrict server access to authorized subnets only, while keeping HTTP/HTTPS traffic flowing to the internet.

NAT & PAT Configuration

A BGP session between two routers keeps dropping — the peer is up one minute, gone the next. You'll diagnose the flapping session, fix the root cause, configure prefix-lists to control what gets advertised, and verify end-to-end reachability.
